PaperSearch — semantic retrieval in Oracle Database 23ai

Live demo: natural-language queries are embedded locally, then VECTOR_DISTANCE runs inside Oracle on a VECTOR(384, FLOAT32) column backed by an optional HNSW index (FETCH APPROXIMATE when available). Optional hybrid filters (year range + substring on arXiv categories) are pushed into SQL as plain predicates alongside vector ordering.
